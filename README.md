# Vulnerability Assessment Report

## Internship
Cyber Security Internship – Future Interns

## Task
Task 1 – Vulnerability Assessment Report for a Live Website (Read-Only Scope)

## Target Website
http://testphp.vulnweb.com

## Scope
Passive vulnerability assessment of public-facing pages only.  
No exploitation or intrusive testing was performed.

## Tools Used
- Nmap
- OWASP ZAP (Passive Scan)
- Browser DevTools
- SecurityHeaders.com

## Methodology
1. Performed port and service scanning using Nmap.
2. Conducted passive vulnerability analysis using OWASP ZAP.
3. Analyzed HTTP security headers.
4. Inspected cookies and client-side configurations using browser developer tools.
5. Documented findings and classified risk levels.
